## Lumen Starter Rest API 
This project is a simple example of how to use lumen to create a rest api.

## Features
- JWT Authentication
- Dashboard
- CRUD Category
- CRUD Order & Order Detail
- Change Password

## Tech Stack
- PHP 8.1
- Lumen 10.0
- JWT 2.0
- MySQL

## Instalation
    php artisan migrate --seed
    php artisan jwt:secret

## Run
    php -S localhost:7000 -t public
